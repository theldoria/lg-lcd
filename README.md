# LgLcd

Ruby binding to the 'Logitech LCD SDK Package'

## Installation

Add this line to your application's Gemfile:

    gem 'lg-lcd'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lg-lcd
